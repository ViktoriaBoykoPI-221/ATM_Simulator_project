Проект "Симуляция банкомата" реализует программу-имитацию банкомата, которая позволяет 
пользователям вносить, снимать и переводить деньги. Основная цель проекта - создание 
единственного экземпляра банкомата с помощью паттерна Singleton, чтобы гарантировать, что 
будет существовать только один функционирующий банкомат и предотвратить создание 
нескольких параллельных банкоматов.

Описание использования паттерна Singleton:
1. Строки 5-18: В классе `ATM` определен приватный конструктор и статическое приватное поле 
`instance`, которое хранит единственный экземпляр класса.
2. Строка 13: Метод `Instance` является статическим свойством, возвращающим единственный 
экземпляр класса `ATM`. Если экземпляр еще не создан, то создается новый экземпляр и 
возвращается сохраненная ссылка на него.
3. Строки 51-53: В методе `Main` создается единственный экземпляр банкомата с помощью 
вызова `ATM atm = ATM.Instance`.

Паттерн Singleton гарантирует, что будет создан только один экземпляр класса `ATM`, и все 
операции с банкоматом будут выполняться через этот экземпляр.
